# E-Commerce Application

## Mô tả
Dự án xây dựng ứng dụng bán hàng sử dụng **Spring Boot** cho Backend và **Angular** cho Frontend.

Backend cung cấp **RESTful API** phục vụ các chức năng chính của hệ thống bán hàng, bao gồm:
- Đăng ký và đăng nhập người dùng
- Quản lý người dùng
- Quản lý sản phẩm
- Quản lý đơn hàng

Hệ thống xác thực và phân quyền được xây dựng dựa trên **Spring Security** và **JWT (JSON Web Token)**.

## Công nghệ sử dụng
### Backend
- Java Spring Boot
- Spring Security
- JWT
- Spring Data JPA
- Maven

### Frontend
- Angular
- TypeScript
- HTML / SCSS
- Bootstrap
